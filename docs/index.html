How to:

1. Make sure, you're time synced to the required server, if you do know the specific time server.

2. Your browser will most likely try to deny more than one pop-up window, to avoid this behavior, make sure to add at least two test pages and click the run test button, which should open just one pop-up window, you may freely close this window and return back to this page. Next to the address bar of your browser you should see a warning, that all the pop-ups got blocked, make sure to allow the page to launch multiple pop-ups, then you're good to go! If you still can't figure this out, feel free to check out Chrome and Firefox guides.
